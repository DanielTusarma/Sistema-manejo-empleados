# Sistema de manejo de empleados

Aplicación fullstack con Python y Angular para la gestión de empleados utilizando arquitectura cliente-servidor y APIREST.

## Tecnologías

### Backend
- Python
- Flask
- SQLAlchemy (ORM)
- Flask-Migrate (migraciones)
- PyMySQL
- Marshmallow (serialización)
- Flask-CORS
- MySQL

### Frontend
- Angular
- Bootstrap (UI)

## Descripción

- Sistema para la gestión de empleados mediante una API REST.
- Actualmente permite el registro y consulta de empleados.

## Arquitectura

- API REST construida con Flask
- Base de datos MySQL
- Frontend desacoplado en Angular
